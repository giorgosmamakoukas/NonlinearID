Instructions for heat exchanger example.

Dependencies:

Yalmip
Use: setting up SDP/SOS programs for LREE & iEE.
Available: https://yalmip.github.io/

Mosek
Use: solver for SDP/SOS programs.
Available: https://www.mosek.com/

Spotless
Use: setting up SDP/SOS programs LREE, LRSE & iEE. 
See also: https://github.com/imanchester/NonlinearID  for further instructions on the use of spotless with the specialized algorithm for LRSE.
Available: https://github.com/spot-toolbox/spotless

Specialized Algorithm for LRSE
Use: required for LRSE. Also contains functions for specifying models in LREE and iEE.
Available: https://github.com/imanchester/LRSE


Instructions:

Run daISy_heatExchanger_example.m to generate results in Section V.B of the paper.
